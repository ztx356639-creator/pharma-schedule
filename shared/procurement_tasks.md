# 采购部任务参考清单

> 用于 --test-items-json 的 department: "procurement" 部分

## 原辅料采购

| 任务 | 耗时 | 优先级 | 备注 |
|------|------|--------|------|
| 物料需求确认 | 0.5天 | 高 | 根据批生产指令核对 |
| 供应商询价 | 1天 | 高 | 至少3家比价 |
| 采购下单 | 0.5天 | 高 | 含合同签订 |
| 供应商发货跟踪 | 3-7天 | 普通 | 视供应商距离 |
| 原辅料到货验收 | 0.5天 | 高 | 核对品名、批号、数量、COA |
| 入库（待检区） | 0.5天 | 高 | 贴待检标签 |

## 包材采购

| 任务 | 耗时 | 优先级 | 备注 |
|------|------|--------|------|
| 包材需求确认 | 0.5天 | 普通 | 铝箔、PVC/PVDC、说明书、标签 |
| 包材下单 | 0.5天 | 普通 | 定制包材需提前更多 |
| 包材到货验收 | 0.5天 | 高 | 核对文字内容、印刷质量 |
| 包材入库 | 0.5天 | 普通 | |

## 试剂/耗材采购

| 任务 | 耗时 | 优先级 | 备注 |
|------|------|--------|------|
| QC试剂需求确认 | 0.5天 | 普通 | 对照检验方法 |
| 试剂下单 | 0.5天 | 普通 | 标准品、对照品需提前 |
| 试剂到货验收 | 0.5天 | 普通 | 核对批号、效期、COA |

## 关键注意

- 原辅料采购周期通常 7-15 天，定制包材 15-30 天
- 进口原辅料需考虑海关时间（额外 7-14 天）
- 特殊管制物料需提前办理相关手续
- 供应商资质需定期审计，过期需重新审计
