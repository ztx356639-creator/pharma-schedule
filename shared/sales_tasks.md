# 销售发货任务参考清单

> 用于 --test-items-json 的 department: "sales" 部分

## 放行与发货

| 任务 | 耗时 | 优先级 | 备注 |
|------|------|--------|------|
| 放行资料整理 | 0.5天 | 高 | 批记录、检验报告、偏差记录 |
| QA放行审核 | 1天 | 高 | QA经理签字 |
| 成品入库（合格品库） | 0.5天 | 高 | 从待验区转入 |
| 发货计划制定 | 0.5天 | 普通 | 按客户订单排优先级 |
| 物流安排 | 0.5天 | 普通 | 冷链/常温、运输条件确认 |
| 出库发货 | 0.5天 | 高 | 核对品名、批号、数量 |
| 运输途中监控 | 1-3天 | 普通 | 温度监控（如需） |
| 客户签收确认 | 1-3天 | 普通 | |

## 特殊情况

| 任务 | 耗时 | 优先级 | 备注 |
|------|------|--------|------|
| 出口药品注册资料 | 3-5天 | 高 | 如需出口 |
| 冷链运输验证 | 1天 | 高 | 生物制品等 |
| 样品寄送 | 0.5天 | 普通 | 客户留样 |

## 关键注意

- 成品检验合格 + QA放行签字后才能发货
- 运输条件需与药品储存条件一致
- 冷链药品需全程温度监控并记录
- 发货后需在规定时间内完成签收确认
- 留样需按法规要求保存至效期后一年
