# QC检验任务参考清单

> 用于 --test-items-json 的 department: "qc_incoming" / "qc_finished" 部分

## 来料检验（qc_incoming）

| 任务 | 仪器 | 耗时 | 优先级 | 适用 |
|------|------|------|--------|------|
| 取样 | 取样工具 | 0.5天 | 高 | 所有 |
| 性状检查 | 目视 | 0.5天 | 普通 | 所有 |
| 鉴别-化学 | 通风橱 | 1h | 普通 | 原料药 |
| 鉴别-红外(IR) | 红外光谱仪 | 1h | 普通 | 原料药 |
| 鉴别-紫外(UV) | 紫外分光光度计 | 1h | 普通 | 原料药 |
| 鉴别-HPLC | HPLC | 2h | 普通 | 有含量项的 |
| 有关物质 | HPLC | 4h | 高 | 原料药 |
| 水分 | Karl Fischer | 1h | 普通 | 固体 |
| 干燥失重 | 分析天平+干燥箱 | 4h | 普通 | 原料药 |
| 含量测定 | HPLC/UV/滴定 | 2-3h | 高 | 所有 |
| 残留溶剂 | GC | 3h | 高 | 原料药 |
| 重金属 | 通风橱 | 2h | 高 | 原料药 |
| 微生物限度 | 生物安全柜 | 5天 | 高 | 非无菌 |
| 出具检验报告 | — | 0.5天 | 高 | 所有 |

## 成品检验（qc_finished）

| 任务 | 仪器 | 耗时 | 优先级 | 适用 |
|------|------|------|--------|------|
| 取样 | 取样工具 | 0.5天 | 高 | 所有 |
| 性状 | 目视 | 0.5天 | 普通 | 所有 |
| 鉴别（多项） | IR/UV/HPLC | 1-2天 | 高 | 所有 |
| 有关物质 | HPLC | 4h | 高 | 大部分 |
| 溶出度 | 溶出仪+UV/HPLC | 6h | 高 | 口服固体 |
| 崩解时限 | 崩解仪 | 1h | 普通 | 片剂/胶囊 |
| 水分 | Karl Fischer | 1h | 普通 | 固体 |
| 含量测定 | HPLC/UV | 3h | 高 | 所有 |
| 微生物限度 | 生物安全柜 | 5天 | 高 | 非无菌 |
| 无菌检查 | 无菌室 | 14天 | 高 | 无菌制剂 |
| 不溶性微粒 | 粒子计数器 | 1h | 高 | 注射剂 |
| 可见异物 | 灯检台 | 1h | 高 | 注射剂 |
| 装量差异 | 天平 | 1h | 普通 | 液体制剂 |
| 出具检验报告 | — | 0.5天 | 高 | 所有 |

## HPLC进样序列优化

可合并进样节省时间：
- 鉴别-HPLC + 有关物质 + 含量测定 → 同一个HPLC序列
- 先做系统适用性 → 再进样品 → 最后做结果计算
- 合并后预计 1 天完成（vs 分开做 2-3 天）

## 关键注意

- 微生物限度需提前安排（5天培养周期）
- 无菌检查需提前安排（14天培养周期）
- HPLC 方法切换需 30min 平衡/冲洗
- 标准品/对照品需确认效期
- OOS（超标结果）需启动调查流程
