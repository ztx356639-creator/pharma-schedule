# 药厂 QC 领域知识速查

> 来源：用户（药厂 QC 背景）+ GMP 实践。用于排班时参考检验项目、仪器、耗时。

## 常见检验项目 × 仪器 × 耗时

| 检验项目 | 常用仪器 | 耗时 | 适用 |
|----------|----------|------|------|
| 性状 | 目视 | 0.5h | 所有 |
| 鉴别-化学 | 通风橱、试管 | 1h | 固体制剂 |
| 鉴别-红外(IR) | 红外光谱仪 | 1h | 原料药/固体制剂 |
| 鉴别-紫外(UV) | 紫外分光光度计 | 1h | 原料药 |
| 鉴别-HPLC | HPLC | 2h | 有HPLC含量项的 |
| 有关物质 | HPLC | 4h | 原料药/注射剂 |
| 溶出度 | 溶出仪+UV/HPLC | 6h | 口服固体制剂 |
| 崩解时限 | 崩解仪 | 1h | 片剂/胶囊 |
| 水分 | Karl Fischer | 1h | 固体制剂 |
| 干燥失重 | 分析天平+干燥箱 | 4h+ | 原料药 |
| 重金属 | 通风橱 | 2h | 原料药 |
| 残留溶剂 | GC(顶空) | 3h | 原料药 |
| 微生物限度 | 微生物实验室 | **5天** | 非无菌制剂 |
| 细菌内毒素 | 内毒素检测仪 | 2h | 注射剂 |
| 含量-HPLC | HPLC | 3h | 大部分制剂 |
| 含量-UV | 紫外分光光度计 | 2h | 部分制剂 |
| 含量-滴定 | 滴定管 | 1.5h | 原料药 |
| 无菌检查 | 无菌室 | **14天** | 无菌制剂 |
| 不溶性微粒 | 粒子计数器 | 1h | 注射剂 |
| 可见异物 | 灯检台 | 1h | 注射剂 |

## 刚性约束（不可压缩）

- **微生物限度**：培养 5 天，必须提前安排
- **无菌检查**：培养 14 天，必须提前安排
- **稳定性考察**：长期/加速/中间条件，按月计
- **方法验证**：精密度/准确度/线性/耐用性，比放行检验多 2-3 倍时间
- **方法转移**：需要双方实验室同步，耗时更长

## 仪器方法切换

- HPLC 换方法：30min 平衡/冲洗
- GC 换方法：20-30min 平衡
- UV 换波长：5-10min
- 溶出仪换介质：15-20min

## 仪器校准周期（常见）

| 仪器 | 校准周期 | 备注 |
|------|----------|------|
| HPLC | 12个月 | 含泵、检测器、柱温箱 |
| UV | 12个月 | 波长+吸光度 |
| GC | 12个月 | FID/TCD |
| 分析天平 | 12个月 | 外校 |
| pH 计 | 6个月 | 用标准液校准 |
| 溶出仪 | 12个月 | 机械验证 |
| 红外光谱仪 | 12个月 | |

## GMP 关键节点

1. **原料未放行，车间不得投料**
2. **成品未放行，销售不得发货**
3. **偏差必须记录**：检验过程中任何 OOS/OOT 需走偏差流程
4. **留样**：每批留样，留样量 ≥ 2 倍全检量
5. **记录及时**：检验记录必须同步完成，不能事后补

## 常见仪器冲突场景

- HPLC 是最紧张的资源：鉴别+有关物质+含量都用，一台 HPLC 一天最多跑 2-3 个序列
- 微生物和理化通常在不同实验室，不冲突
- 溶出仪占用时间长（6h+），一天最多排 1-2 个品种
- GC 顶空进样单次时间长，残留溶剂检验排期需注意
